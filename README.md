# 📊 Retail Sales Forecasting & Demand Intelligence Dashboard

An end-to-end Retail Sales Forecasting project developed using Python and Streamlit. This project analyzes historical retail sales data, predicts future sales trends, detects anomalies, and segments products based on demand using machine learning techniques.

Author: Yamini Nebhnani  
Project: Retail Sales Forecasting

---

# 🎯 Project Objective

The objective of this project is to analyze retail sales data, forecast future demand, identify unusual sales patterns, and provide business insights through an interactive Streamlit dashboard.

---

| Category | Technology |
|----------|------------|

| Data Analysis | Pandas, NumPy |
| Visualization | Matplotlib |
| Machine Learning | Scikit-learn |

| IDE | Google Colab, VS Code |
| Version Control | GitHub |

---

# 🚀 Features

- 📈 Interactive Sales Dashboard
- 📊 Sales Overview with KPIs
- 📅 Monthly & Yearly Sales Analysis
- 🔮 Sales Forecast Explorer
- ⚠️ Anomaly Detection using Isolation Forest
- 🧩 Product Demand Segmentation using K-Means Clustering
- 📉 PCA Visualization for Product Clusters
- 📄 Business Summary Report

# ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Retail-Sales-Forecasting.git
```

### Install Libraries

```bash
pip install -r requirements.txt
```

### Run Streamlit

```bash
streamlit run app.py
```
